# Network Traffic Analysis using Wireshark

This project is completed as part of my Cybersecurity Internship.

## Objective
- To understand network traffic behavior
- To analyze packets using Wireshark
- To identify suspicious network activity
- To gain hands-on experience in traffic monitoring

## Tools Used
- Wireshark
- Python
- Scapy Library
- GitHub

## Description
This project focuses on capturing and analyzing network packets to understand protocols such as TCP, UDP, HTTP, and DNS. Basic packet analysis helps identify abnormal or suspicious traffic patterns.

## Analysis Performed
- Packet capture
- Protocol filtering
- Source and destination IP analysis
